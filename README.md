# Shopzy 🛍️

Shopzy is a modern full-stack MERN Ecommerce web application built using MongoDB, Express.js, React.js, and Node.js.

🚀 Live Demo: https://shopzy-puce.vercel.app

---

# ✨ Features

- 🔐 User Authentication (JWT Login/Register)
- 🛒 Add To Cart & Remove From Cart
- ⚡ Buy Now Feature
- 📦 Checkout System
- 💳 Payment Method Selection
- 🔍 Product Search
- 📱 Fully Responsive UI
- ☁️ Full Stack Deployment
- 📦 Orders Management
- 🚀 REST API Integration

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Tailwind CSS
- React Router DOM
- Axios
- React Toastify

## Backend
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose
- JWT Authentication

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/santhosh-103/Shopzy.git
```

---

# Backend Setup

```bash
cd server
npm install
npm start
```

---

# Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

# 🌐 Environment Variables

Create a `.env` file inside the `server` folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```
